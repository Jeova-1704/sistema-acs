**ACs UPE - Arquitetura**
## Descrição
Este artefato pretende documentar a arquitetura do sistema de atividades complementares da UPE (**ACs-UPE**). Ele apresenta uma visão geral de arquitetura abrangente do sistema e serve como um meio de comunicação para membros da equipe do projeto, com relação a decisões arquiteturalmente significativas tomadas sobre o projeto.

## O que é o ACs-UPE?
O ACs-UPE é um sistema online que oferece uma plataforma web para que os alunos possam enviar suas requisições de atividades complementares e acompanhar o fluxo de avaliação realizada pelos membros avaliadores. Além disso, também proporciona aos avaliadores uma maneira mais eficiente de realizar as avaliações de ACs e melhorar o gerenciamento das requisições entre eles.
